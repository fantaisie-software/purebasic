@Title ASCII-Codes

@Section Die ASCII-Codes 0 bis 255 und ihre dazugehörigen Zeichen

  Zur Information: Die folgende ASCII-Tabelle ist die in Westeuropa verwendete Tabelle.
  Zeichen von 0 bis 127 sind US-ASCII-Standard und Zeichen von 128 bis 255 sind ISO 8859-1,
  dessen vollständiger Name ISO/IEC 8859-1 lautet und häufig als Latin-1 oder Westeuropa
  (Western Europe) bezeichnet wird und Windows-1252-kompatibel ist (CP-1252).

@Section Erweiterte Beschreibungen der wichtigsten Anwendungen von Kontrollzeichen (ASCII Schriftzeichen 0 bis 31)

@FormatIf HTML WEBSITE
  <table border bgcolor="#FFFFFF" width="86%">
  <tr>
  <td><b>ASCII</b></td>

  <td><b>Beschreibung</b></td>
  </tr>
  <tr bgcolor="#FFFFCC"><td>0</td><td><font size=-1>Null-Zeichen, normalerweise (und insbesondere in PureBasic) verwendet, um das Ende des Strings anzuzeigen.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>1</td><td><font size=-1>Anfang des Headers (Kopfes). Markiert den Anfang eines Blocks von Daten oder den Kopfteil eines Blocks von Daten an.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>2</td><td><font size=-1>Beginn eines Textes. Wird normalerweise als erstes Zeichen in einem Block mit Text gesendet, während Datenübermittlungen.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>3</td><td><font size=-1>Ende eines Textes. Wird normalerweise als letztes Zeichen in einem Block mit Text gesendet, während Datenübermittlungen.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>4</td><td><font size=-1>Ende des Headers (Kopfes). Wird verwendet, um das Ende einer Übertragung anzuzeigen.</font></td></tr>

  <tr bgcolor="#FFFFFF"><td>5</td><td><font size=-1>Anfrage. Wird an einen Empfänger gesandt, um eine Antwort zu bekommen.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>6</td><td><font size=-1>Bestätigen. Wird von einem Empfänger gesandt, um zu zeigen, dass er die Bitte/Anforderung erhalten und/oder verstanden hat.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>7</td><td><font size=-1>Glocke. Erzeugt ein hörbares Geräusch (normalerweise gibt der PC-Lautsprecher ein 'Beep' aus).</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>8</td><td><font size=-1>Backspace (Rücktaste). Bewegt den Cursor um eine Position nach links (könnte auch das Zeichen nach links vor der Bewegung löschen).</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>9</td><td><font size=-1>Tab. Waagerechter Tabulator, normalerweise verwendet, um text-basierte Tabellen zu formatieren.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>10</td><td><font size=-1>Zeilenvorschub. Das Zeichen wird zur Darstellung der Funktion verwendet, sich eine Zeile auf einer Schreibmaschine oder einem Textmodusdrucker nach vorn (unten) zu bewegen. Normalerweise verwendet als das - oder Teil des - 'End of Line' (Ende der Zeile) Zeichens.</font></td></tr>

  <tr bgcolor="#FFFFFF"><td>11</td><td><font size=-1>Senkrechter Tabulator. Genauso wie der (waagerechte) Tabulator, bewegt sich aber eine Zeile/Reihe nach unten, anstatt über eine Spalte.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>12</td><td><font size=-1>Formularvorschub. Normalerweise verwendet als das Zeichen, um einen (Textmodus) Drucker aufzufordern, sich auf die nächste Seite (Formular) zu bewegen.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>13</td><td><font size=-1>Wagenrücklauf. Das Zeichen, welches den Kopf einer Schreibmaschine oder eines Druckers auffordert, sich zurück zum Anfang einer Zeile zu bewegen. Normalerweise verwendet als das - oder Teil des - 'End of Line' (Ende der Zeile) Zeichens.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>14</td><td><font size=-1>Shift out (ausrücken). Startet einen Block von Zeichen, deren Bedeutung abhängig von der Anwendung ist.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>15</td><td><font size=-1>Shift in (einrücken). Beendet die Übertragung des oben genannten Block-Typs.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>16</td><td><font size=-1>Data Link Escape (Datenübertragung-Escape). Wird verwendet, um anzuzeigen, dass das folgende Kontrollzeichen als Daten anstatt als Kontrollzeichen interpretiert werden soll.</font></td></tr>

  <tr bgcolor="#FFFFFF"><td>17</td><td><font size=-1>Device Control 1. Normalerweise verwendet, um ein Ausrüstungsteil einzuschalten. Die häufigste Verwendung ist heute als XON Zeichen im durch Software kontrollierten Ablauf serieller Kommunikationen.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>18</td><td><font size=-1>Device control 2. Ein weiteres situationsabhängiges Device-Control Zeichen.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>19</td><td><font size=-1>Device control 3. Normalerweise verwendet, um ein Ausrüstungsteil auszuschalten. Die häufigste Verwendung ist heute als XOFF Zeichen im durch Software kontrollierten Ablauf serieller Kommunikationen.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>20</td><td><font size=-1>Device control 4. Ein weiteres situationsabhängiges Device-Control Zeichen.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>21</td><td><font size=-1>Negative Bestätigung. Normalerweise verwendet, um nicht empfangene oder nicht verstandene (fehlerhafte) Daten anzuzeigen.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>22</td><td><font size=-1>Synchroner Leerlauf (Idle). Wie der Name andeutet ein Signal, das synchron (zu periodischen Zeiten) gesandt wird, um anzuzeigen, dass der Verbindungskanal leer, aber noch aktiv ist.</font></td></tr>

  <tr bgcolor="#FFFFFF"><td>23</td><td><font size=-1>Ende des Übertragungsblocks. Verwendet, um die Übertragung von Daten durch Anzeigen eines Blockendes zu kontrollieren. Dies ist nicht das Gleiche wie EOT.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>24</td><td><font size=-1>Abbruch. Bedeutet normalerweise, dass zuvor gesandte Daten ignoriert werden sollten, obwohl Details abhängig von der Anwendung sind.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>25</td><td><font size=-1>Ende des Mediums. Wird verwendet, um das Ende irgendwelcher Medien, zum Beispiel das Ende eines Bandlaufwerks, anzuzeigen.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>26</td><td><font size=-1>Ersatz (Substitute). Ein Platzhalter-Zeichen, das zum Anzeigen verwendet wird, dass ein Zeichen ersetzt worden ist.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>27</td><td><font size=-1>Escape-Zeichen. Normalerweise wird das Zeichen durch Drücken der 'Esc'-Taste erzeugt, es wird in &quot;Escape-Sequenzen&quot; verwendet, um textbasierten Ausgaben (Konsole, Drucker, etc.) Formatinformationen bieten zu können.</font></td></tr>

  <tr bgcolor="#FFFFCC"><td>28</td><td><font size=-1>Datei-Trennzeichen.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>29</td><td><font size=-1>Gruppen-Trennzeichen.</font></td></tr>
  <tr bgcolor="#FFFFCC"><td>30</td><td><font size=-1>Datensatz (Record) Trennzeichen.</font></td></tr>
  <tr bgcolor="#FFFFFF"><td>31</td><td><font size=-1>Einheiten (Unit) Trennzeichen.</font></td></tr>
  </table>
