@Library Gadget3D

@Overview

  Die Gadget3D Bibliothek ermöglicht die Erstellung komplexer grafischer Bedienoberflächen (englisch "Graphical 
  User Interface" - kurz "GUI") direkt über dem Screen-Bereich, mittels der 3D-Engine. Dies ist hauptsächlich
  gedacht für Spiele und Anwendungen, welche Anwendereingaben benötigen, während sie im Vollbildschirm-Modus
  laufen. Diese Bibliothek basiert auf der regulären PureBasic @LibraryLink "gadget" "Gadget-Bibliothek", und
  bietet ähnlichen Syntax und Verhalten. Die verwendete GUI-Engine ist CEGUI, welche einige schöne Optionen
  anbietet - wie "Skinning", gute Geschwindigkeit und eine Menge an eingebauten Gadgets. Mehr Informationen
  über CEGUI sind hier zu finden: @InternetLink "http://www.cegui.org.uk" "http://www.cegui.org.uk".
@LineBreak
@LineBreak
  Sie verwendet die 3D-Engine, daher muss vor dem Verwenden dieser Funktionen 
  @@InitEngine3D erfolgreich aufgerufen worden sein.
@LineBreak
@LineBreak
  Vor dem Verwenden von Gadgets wird normalerweise zuerst ein @Link "Window3D/OpenWindow3D" "Fenster" geöffnet.

@CommandList

@ExampleFile All Gadget3D.pb

@SupportedOS

@Function Ergebnis = AddGadgetItem3D(#Gadget3D, Position, Text$)

@Description
  Fügt einen Eintrag zum angegebenen #Gadget3D hinzu.
  
@Parameter "#Gadget3D"
  Das zu verwendende 3D-Gadget.
  
@Parameter "Position"
  Der Eintrag-Index, wo der neue Eintrag eingefügt werden
  soll. Um diesen Eintrag am Anfang einzufügen, benutzen Sie einen Index von 0.
  Um diesen Eintrag am Ende der aktuellen Eintrag-Liste hinzuzufügen, benutzen
  Sie einen Wert von -1. Denken Sie beim Einfügen eines Eintrags daran, dass alle
  aktuellen Einträge die auf diesen neuen Eintrag folgen, einen um 1 höheren Positions-Index
  erhalten.
  
@Parameter "Text$"
  Der neue Eintrag-Text.
  
@NoReturnValue
  
@Remarks
  Die folgenden Gadgets werden unterstützt:
@LineBreak
@LineBreak
  - @@ComboBoxGadget3D @LineBreak
  - @@ListViewGadget3D @LineBreak
  - @@PanelGadget3D @LineBreak
@LineBreak

@SeeAlso
  @@ComboBoxGadget3D, @@ListViewGadget3D, @@PanelGadget3D

@SupportedOS

@Function Ergebnis = ButtonGadget3D(#Gadget3D, x, y, Breite, Höhe, Text$)

@Description
  Erstellt ein Schalter-Gadget innerhalb der aktuellen Gadget-Liste.
  
@Parameter "#Gadget3D"
  Eine Nummer zum Identifizieren des neuen 3D-Gadgets.

  automatisch zu generieren.

@Parameter "x, y, Breite, Höhe"
  Die Position und Abmessungen des neuen Gadgets.

@Parameter "Text$"
  Der auf dem Schalter darzustellende Text.

@ReturnValue

  als #Gadget3D Parameter verwendet wurde, dann enthält bei Erfolg der Rückgabewert
  die automatisch generierte Nummer.

@Remarks

@LineBreak
@LineBreak
  Die folgenden Funktionen können zum Kontrollieren des Gadgets verwendet werden: @LineBreak
  - @@SetGadgetText3D: Ändert den Text des ButtonGadgets. @LineBreak
  - @@GetGadgetText3D: Ermittelt den Text des ButtonGadgets.

@Function Ergebnis = ComboBoxGadget3D(#Gadget3D, x, y, Breite, Höhe [, Flags])

@Description

@Parameter "#Gadget3D"

  die automatisch generierte Nummer.

@Remarks

@LineBreak

@LineBreak
@LineBreak

  - @@SetGadgetItemText3D: Ändert den Text eines Gadget-Eintrags. @LineBreak
@LineBreak
  - @@GetGadgetState3D: Ermittelt den Index (beginnend bei 0) des aktuellen Elements. @LineBreak
  - @@GetGadgetText3D: Ermittelt den (Text-) Inhalt des aktuellen Elements. @LineBreak
  - @@SetGadgetState3D: Ändert das ausgewählte Element. @LineBreak
  - @@SetGadgetText3D: Ändert den anzuzeigenden Text. Wenn das ComboBoxGadget nicht editierbar ist, muss sich der Text in der Aufklapp-Liste befinden.

@Function Ergebnis = ContainerGadget3D(#Gadget3D, x, y, Breite, Höhe)

@Description
  Erstellt ein Container Gadget innerhalb der aktuellen Gadgetliste.
@LineBreak
@LineBreak
  Es ist ein simples Panel-Gadget
  (Schalttafel), welches andere Gadgets enthalten kann. Sobald dieses Gadget erstellt wurde,
  werden alle nachfolgenden Gadgets innerhalb des Containers erstellt. Wenn alle benötigten
  Gadgets erstellt wurden, muss @@CloseGadgetList3D aufgerufen werden, um zur vorherigen
  Gadgetliste zurückzukehren. @@OpenGadgetList3D kann später benutzt werden, um weitere
  Gadgets während des Programmablaufs ("on-the-fly") zum Container-Bereich hinzuzufügen.

@Parameter "#Gadget3D"
  Eine Nummer zum Identifizieren des neuen 3D-Gadgets.
  @ReferenceLink "purebasic_objects" "#PB_Any" kann verwendet werden, um diese Zahl
  automatisch zu generieren.

@Parameter "x, y, Breite, Höhe"
  Die Position und Abmessungen des neuen Gadgets.

@ReturnValue
  Gibt im Erfolgsfall einen Wert ungleich Null zurück, andernfalls Null. Wenn @#PB_Any
  als #Gadget3D Parameter verwendet wurde, dann enthält bei Erfolg der Rückgabewert
  die automatisch generierte Nummer.

@Remarks
  Zu diesem Gadget kann mittels @@GadgetToolTip3D eine 'Mini-Hilfe' hinzugefügt werden.

@SupportedOS

@Function Ergebnis = SpinGadget3D(#Gadget3D, x, y, Breite, Höhe, Minimum, Maximum)

@Description
  Erstellt ein SpinGadget in der aktuellen Gadgetliste.

@Parameter "#Gadget3D"
  Eine Nummer zum Identifizieren des neuen 3D-Gadgets.
  @ReferenceLink "purebasic_objects" "#PB_Any" kann verwendet werden, um diese Zahl
  automatisch zu generieren.

@Parameter "x, y, Breite, Höhe"
  Die Position und Abmessungen des neuen Gadgets.

@Parameter "Minimum, Maximum"
  Die Minimal- und Maximal-Werte, die das SpinGadget annehmen kann.

@ReturnValue
  Gibt im Erfolgsfall einen Wert ungleich Null zurück, andernfalls Null. Wenn @#PB_Any
  als #Gadget3D Parameter verwendet wurde, dann enthält bei Erfolg der Rückgabewert
  die automatisch generierte Nummer.

@Remarks
  Zu diesem Gadget kann mittels @@GadgetToolTip3D eine 'Mini-Hilfe' hinzugefügt werden.
@LineBreak
@LineBreak
  Die folgenden Funktionen können verwendet werden, um auf das Gadget zuzugreifen:
@LineBreak
@LineBreak
  @@GetGadgetState3D: Ermittelt den aktuellen Gadget-Wert. @LineBreak
  @@SetGadgetState3D: Ändert den aktuellen Gadget-Wert. Für das Anzeigen des neuen Werts ist jedoch trotzdem noch @@SetGadgetText3D notwendig! @LineBreak
  @@GetGadgetText3D: Ermittelt den im Gadget enthaltenen Text. @LineBreak
  @@SetGadgetText3D: Ändert den im Gadget enthaltenen Text. @LineBreak
  @@GetGadgetAttribute3D: Mit einem der folgenden Attribute:
@FixedFont
  @#PB_Spin3D_Minimum   : Gibt den Minimal-Wert zurück.
  @#PB_Spin3D_Maximum   : Gibt den Maximal-Wert zurück.
@EndFixedFont

  @@SetGadgetAttribute3D: Mit einem der folgenden Attribute:
@FixedFont
  @#PB_Spin3D_Minimum   : Ändert den Minimal-Wert.
  @#PB_Spin3D_Maximum   : Ändert den Maximal-Wert.
@EndFixedFont

@SupportedOS

@Function Ergebnis = StringGadget3D(#Gadget3D, x, y, Breite, Hoehe, Inhalt$ [, Flags])

@Description
  Erstellt ein String-Gadget innerhalb der aktuellen Gadget-Liste.
@LineBreak
@LineBreak
  Dieses Gadget akzeptiert nur eine Zeile an Text. Um mehrzeilige Eingaben zu erhalten,
  verwenden Sie das @@EditorGadget3D.

@Parameter "#Gadget3D"
  Eine Nummer zum Identifizieren des neuen 3D-Gadgets.
  @ReferenceLink "purebasic_objects" "#PB_Any" kann verwendet werden, um diese Zahl
  automatisch zu generieren.

@Parameter "x, y, Breite, Höhe"
  Die Position und Abmessungen des neuen Gadgets.

@Parameter "Inhalt$"
  Der anfängliche Text-Inhalt.

@OptionalParameter "Flags"
  Dies kann eine Kombination (verknüpft mit dem '|' OR-Operator) der folgenden Werte sein:
@FixedFont
  @#PB_String3D_Numeric    : Nur (positive) Ganzzahlen werden akzeptiert.
  @#PB_String3D_Password   : Passwort-Modus, es werden nur '*' anstelle normaler Zeichen angezeigt.
  @#PB_String3D_ReadOnly   : 'Read only' Modus. Es kann kein Text eingegeben werden.
@EndFixedFont

@ReturnValue
  Gibt im Erfolgsfall einen Wert ungleich Null zurück, andernfalls Null. Wenn @#PB_Any
  als #Gadget3D Parameter verwendet wurde, dann enthält bei Erfolg der Rückgabewert
  die automatisch generierte Nummer.

@Remarks
  Der Inhalt kann später mit @@SetGadgetText3D verändert und mit @@GetGadgetText3D
  ermittelt werden.
@LineBreak
@LineBreak
  Die Abfrage der folgenden Ereignisse wird mittels @@EventType3D unterstützt:
@FixedFont
  @#PB_EventType3D_Change    : Der Text wurde durch den Anwender geändert.
  @#PB_EventType3D_Focus     : Das StringGadget erhielt den Fokus.
  @#PB_EventType3D_LostFocus : Das StringGadget verlor den Fokus.
@EndFixedFont

  Zu diesem Gadget kann mittels @@GadgetToolTip3D eine 'Mini-Hilfe' hinzugefügt werden.

@SupportedOS
